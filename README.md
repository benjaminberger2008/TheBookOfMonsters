# The Book of Monsters
C# text based RPG inspired by undertale
<img width="1917" height="1079" alt="Screenshot 2026-06-24 124313" src="https://github.com/user-attachments/assets/cd4f5530-277e-4b70-91c5-e5691df183ff" />


## Features
Arcade and Adventure mode

Colour gameplay in Console

Saving and Loading

Sprite Editor

## Controls
WASD - Navigate Menus and minigames

RETURN - Select option

## Adventure Mode
A story-driven experience following your progress against monsters. 

Features a global kindness mechanic, complex story and a speedrunning tier system.

## Arcade Mode
A fast-paced survival gauntlet built for pure combat. 

Features persistent tracking of your overall runs, current win-streaks, maximum win-streaks, and a live-updating K/D ratio.

Screen flickering is a known issue.
